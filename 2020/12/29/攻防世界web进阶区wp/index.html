攻防世界web进阶区简记

发表于 2020-12-29 更新于 2021-03-10 分类于 攻防世界web

菜鸡做完题就需要记录以防下次忘记。

1. web baby :

​ 提示想想初始页面是哪个，那肯定是index.php嘛，但是页面会自动跳转到1.php,所以我们用burp抓包即可。

2. Training-WWW-Robots :

​ 访问robots.txt即可

3. Web_python_template_injection :

​ 见SSTI模板注入，模板注入真机”模板注入”，因为我根本看不懂别人的payload。

4. php_rce :

​ ThinkPHP5连着好几个版本都有远程执行命令的漏洞，在网上查了一下payload，即

​ /index.php?s=index/think\app/invokefunction&function=call_user_func_array&vars[0]=system&vars[1][]=cat /flag。

​ 以后碰上这个版本的ThinkPHP5都能用。

5. Web_php_include :

​ 文件包含，过滤了”php://“，最简单的方法是直接大小写绕过，但由于我对各种协议一点都不了解，所以我学了另一种。题目里还有一个变量hello，会回显到网页前端，

6. ics-06 :

​ 进去只有一个页面能点开，可以看见url框里有一个?id=1，一开始以为是个sql但干了半天啥回显都没有，看来wp发现是爆破id，立马放进burp跑。

7. warmup :

​ 代码审计题，进去看网页源代码发现source.php，然后看了看直接/../../../../../../../../../../穿出去拿到flag。

8. NewsCenter :

​ 普普通通一道sql。

9. NaNNaNNaNNaN-Batman :

​ 拿到附件之后打开发现是js，将后缀改为html打开发现是搜索框，用记事本打开是乱码，查资料（☞别人的wp）发现是因为eval函数里没执行代码导致的，将eval改为alert后再次打开可以看到源码，算出payload，放入搜索框拿flag。

10. web2 :

​ 逆向一个加密算法，这真的是一个web题？？？

11. PHP2 :

​ 看到题目名字就进index.phps里看了下，果然有源码，把admin两次url加密传进去即可。但我是个傻逼，index.phps是源码页面，不会有php代码的执行结果，执行结果要到index.php里看。呜呜呜，Noah太强辣！！！

12. unserialize3 :

​ 反序列化绕过__wakeup()函数，只需要把属性值改大即可。

13. upload1 :

​ 文件上传，要求上传图片，先写个一句话木马，改后缀成.jpg提交上去，burp拦截下来把后缀改回.php，回显出文件被上传到的位置，拿蚁剑连接即可拿到flag。

14. Web_php_unserialize :

​ 反序列化要绕过正则和__wakeup()，要把O:替换成O:+绕过正则，并且base64编码要在php里编码，而不是手动增加%00之后编码。

15. easytornado :

在Tornado里，应用的设置可以通过handler.settings访问

直接拿到secret_cookie，按照题目的方式来加密即可。

16. shrine :

当payload过滤掉了左右小括号，config，self时怎么办呢

17. mfw :

​ git源码泄漏，用GitHacker一扫，拿到网页源码，构造payload拿到flag。

-------------结束啦!感谢阅读!-------------
